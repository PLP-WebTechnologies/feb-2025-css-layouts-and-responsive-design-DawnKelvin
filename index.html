Responsive Web Design Overview

Responsive web design ensures that your website adapts seamlessly to different screen sizes and orientations providinga user friendly experience on all devices.

It is an important user experience design principle, making sure your site works across multiple devices.

The core design concepts for responsive web design include:

  • Fluid Layout: Uses relative units like percentages to make layouts adjust dynamically.
  • Fexible Media: Make videos, images and other media scale proportionally to fit their containers.
  • Media Queries: Makes it possible to apply styles based on specific conditions such as the screen size, resolution or orientation of the device. They are conditional CSS rules.
  • Mobile-First Design: Start designing for smaller screens first then add styles for larger screens using media queries.
  • Flexbox Layout: It aligns, distribute and space elements within a container.

Flexbox Properties
Property Description/Attributes
display: flex Activates flexbox
flex-direction: row|column|row-reverse|column-reverse Sets Layout direction
justify-content Aligns content horizontally
align-items: Aligns content vertically
flex-wrap: Wraps content to next line if it doesn't fit in one line
flex-grow: Allows items to grow and take available space
Grid Layout Properties
Property Description/Attributes
display: grid Activates grid layout
grid-template-columns Define columns structure
grid-template-rows Define rows structure
grid-gap Adds spaces between grid items
grid-template-areas Defines areas in grid layout

Media Queries

Media queries are conditional CSS rules that enable you to apply CSS styles based on specific conditions such as screen sizes, resolution or orientation of the device. They ask the device for information such as its width and apply styles if the conditions are met.

Syntax: @media(condition){/*CSS rule*/}

Types of Media Queries:

  1. Max-Width Media Query: Target screens smaller than specified width.
    Example:
    @media(max-width:760px){ p{ font-size: 18px; } }
  2. Min-Width Media Query: Targets screens larger than a specific width.
    Example:
    @media(min-width:760px){ p{ font-size: 18px; } }
  3. Combined Media Queries: Combines multiple Conditions with logical operators such as AND,OR, NOT.
    Example: @media(Min-Width: 600px) and (orientation:portrait){ p{ font-size:16px; } }

Common Media Query Conditions:

  1. Width and height: Target devices based on their viewport dimensions.
    Example:
    @media(max-width: 480px){ /*Phones*/ }
    @media(min-width: 481px) and (max-width: 1024px){ /*Tablets*/ }
    @media(min-width: 1025px){ /*Desktops*/ }
  2. Orientation: Apply styles based on device orientation.
    Example:
    @media(orientation: landscape){ }
    @media(orientation: portrait){ }
  3. Resolution: Target devices with specific resolutions.
    Example:
    @media(min-resolution: 300dpi){ body{ font-size: 18px; } }
  4. Aspect-ratio: Use aspect ratio for responsive designs e.g., widescreen and square screens.
    Example:
    @media(aspect-ratio:16/9){ video{ width:100%; } }
  5. Hover: Detect devices with hover capabilities e.g., mouse.
    Example:
    @media(hover:hover){ button.hover{ background-color: Orange; } }
